# 🤝 Client Management System

> نظام إدارة العملاء متكامل

## 📋 الوصف
نظام سطح مكتب لإدارة العملاء يشمل بياناتهم،
العقود، المتابعة والتقارير.

## ⚙️ التقنيات المستخدمة
- **Delphi 12** — الواجهة والمنطق البرمجي
- **Microsoft Access** — قاعدة البيانات

## ✨ المميزات
- تسجيل العملاء ومعلومات التواصل
- متابعة العقود والصفقات
- تذكيرات المتابعة
- تقارير وتاريخ العملاء

**نبيل سبع** — مشروع أكاديمي | إعلام آلي وقواعد بيانات
